windows下使用包管理器快速初始化编程环境

支持 win7 及以上的 windows 系统, 将会用到开源第三方包管理器 scoop, 需要PowerShell 5.x以上以及.Net Framework 4.5及以上, 所有的文件都将安装到 C:\Users\Username\scoop\apps目录下

安装 Scoop

打开 PowerShell

运行以下命令:

1
2

Set-ExecutionPolicy RemoteSigned -scope CurrentUser
iwr -useb get.scoop.sh | iex

安装依赖 git 7-zip 版本控制器和解压工具

1

scoop install git 7-zip

安装编程所需的环境

安装 C++ 环境

1

scoop install mingw-winlibs

检查gcc版本

1

gcc -v

安装 Python 环境

1

scoop install python

检查Python版本

1

python -V

安装 Java17 环境

1
2

scoop bucket add java       # 添加java的bucket
scoop install java/openjdk17

检查Java版本

1

java --version

java仍需自己手动配置JAVA_HOME

如果有其它Java版本需求可以在添加bucket之后 使用 scoop search openjdk 检索自己需要的版本之后安装

错误解决

无法以administrator运行

1
2
3
4

PS C:\Users\WDAGUtilityAccount> iwr -useb get.scoop.sh | iex                                                           
Initializing..                                                             
Running the installer as administrator is disabled by default, see https://github.com/ScoopInstaller/Install#for-admin for details.                                                    
Abort.

更换安装命令:

1
2

iwr -useb get.scoop.sh -outfile 'install.ps1'
.\install.ps1 -RunAsAdmin
